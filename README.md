## Coroutines Reinvent

This repository reconstructs coroutine primitives from first principles to gain a precise understanding of their internals and practical trade-offs.
Этот репозиторий для воссоздания котлин корутин с нуля, чтобы изучить их внутреннее устройство досконально и лучше понять плюсы и минусы.

## Getting Started

### Prerequisites

- Java 21
- Gradle 9.2.0
- Kotlin kotlinc-jvm 2.2.20

### Build and Run

- `gradle run`
- Generate the wrapper via `gradle wrapper`, then use `./gradlew run`

## Project Goals

### English

- Rebuild coroutine primitives step by step
- Compare custom implementations with official coroutine behaviors
- Document insights through tests and runnable examples
- Have fun!

### Русский

- Последовательно воссоздать примитивы корутин
- Сравнить собственные реализации с официальным поведением корутин
